Персональная вики
&#160;— тип программного обеспечения, предназначенный для накопления пользовательской информации с использованием
вики-разметки
. Персональная вики позволяет организовать информацию на своём компьютере или мобильном устройстве в форме, подобной
Википедии
, но только для личного использования.
Персональные вики можно разделить на многопользовательские с возможностью организации личного пространства и однопользовательские вики-приложения, не имеющих веб-сервера и сервера баз данных.

MoinMoin
&#160;— функциональный, простой в использовании,
свободный
(
GNU GPL
),
кроссплатформенный
(
Windows
,
Linux
,
Mac OS
), расширяемый вики-движок, написанный на языке
Python
.
TWiki
&#160;— вики-движок для корпоративного использования, написанный на
Perl
.
DokuWiki
&#160;— согласно его автору, простой, но достаточно функциональный вики-движок, который может быть использован для создания любой документации.
WikkaWiki
&#160;— вики-движок, созданный на языке
PHP
с использованием
MySQL
как ответвление заброшенного движка
WakkaWiki
.
Google Sites
&#160;— бесплатный хостинг на базе структурированной вики.
Однопользовательские вики
[
править
|
править код
]
Gnote
&#160;— программа для создания заметок; порт Tomboy, написанный на языке
C++
.
OutWiker
&#91;1&#93;
&#160;— свободная программа под Windows и Linux для создания заметок, в которой каждая заметка хранится в отдельном
каталоге
&#91;2&#93;
.
TiddlyWiki
&#160;— вики-движок и вики-концепция, заключающаяся в том, что весь вики-сайт представляет собой одну HTML-страницу, интерактивность которой обеспечивается
сценариями
.
Tomboy
&#160;— свободная программа для создания заметок (блокнот, использующий вики-разметку), написанная для
Юникс-подобных
ОС на языке
C#
с использованием
Gtk#
, являющаяся частью среды
GNOME
.
WikidPad
&#160;— бесплатная программа, написанная на языке Python, позволяющая хранить списки дел, контакты, заметки и другую информацию с использованием вики-разметки.
Zen-wiki
&#160;— простой вики движок написанный на
python
для
Google App Engine
. Поддерживает синтаксис
Markdown
и подсветку кода. Редактировать вики может только владелец. Также доступен в виде
сервиса
. По состоянию на 25 апреля 2017&#160;— не поддерживается.
Zim
&#160;— текстовый
WYSIWYG
-редактор для создания заметок, написанный на языке Python (версии до 0.29 включительно были написаны на Perl), каждая запись в котором сохраняется в виде текстового файла с вики-разметкой.
didiwiki&#160;— быстрая, легковесная и простая реализация Вики, написанная на языке
C
. Работает на
серверах
Unix
. По умолчанию,
IP
сервера&#160;— 0.0.0.0:8000, но можно изменить на другой, например,
локальный
. Запускается и настраивается из
терминала
, работа с вики-страницами&#160;— из
браузера
.
XOWA
&#160;— копия Википедии на определённую дату, работающая без подключения к интернету.
